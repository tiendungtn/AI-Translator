AI Bilingual Translator - Manual Testing Page

Welcome to the testing page for our translation extension! This page is written in English to let you easily verify if your bilingual translator works correctly. You can test it by clicking the extension icon or using the floating ball in the bottom-right corner.

Quick Note: The extension will translate paragraphs, headings, and list items into your target language (e.g. Vietnamese), displaying the translated text right below the original text!

Section 1: General Paragraphs

Artificial Intelligence has advanced rapidly over the past few years. Modern large language models, like Gemini, can perform translation tasks with human-like understanding, capturing nuances and idioms better than old machine translation engines.

To use the Gemini API, you need to obtain an API key from Google AI Studio. Once saved in the options panel, the extension will send paragraphs in batches to translate them efficiently and display them side-by-side.

Section 2: Headings and Lists

Below is a list of features supported by this bilingual translation extension:

  • Supports both Chrome (Chromium) and Firefox browsers out-of-the-box.
  • Free translation using the public Google Translate API.
  • AI-powered high-quality translation using the Gemini API.
  • Adjustable styles for translation layout, including text color, font size, and line height.
  • Subtle glassmorphic floating action button for quick translation actions.

Section 3: Content to Ignore

The translation script should strictly ignore code blocks and code tags. For example, the following line containing code: const translate = (text) => fetchAPI(text); should only have its outer text translated, leaving the code itself untouched.

Similarly, the entire block of code inside this pre-formatted tag must NOT be translated at all:

function calculateSum(a, b) {
  // This comment should not be translated by default
  const result = a + b;
  return result;
}

Congratulations! If the page layout remains neat and you can read both English and Vietnamese bilingually, the extension is working successfully!
